[DEMO HERE](https://alexandrtovmach.github.io/react-figma-plugin-ds/)

## 🚀 Get Started

Follow these steps to start using React Figma components:

1. Installation

   ```sh
   # with npm
   npm i react-figma-plugin-ds

   # with yarn
   yarn add react-figma-plugin-ds
   ```
